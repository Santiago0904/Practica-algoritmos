1. Operadores de Asignación (Abreviados)
Estos operadores son "atajos" para actualizar el valor de una variable usando su propio valor actual.
+= (Suma y asigna): a += b es lo mismo que a = a + b.
-= (Resta y asigna): a -= b es lo mismo que a = a - b.
*= (Multiplica y asigna): a *= b es lo mismo que a = a * b.
/= (Divide y asigna): a /= b es lo mismo que a = a / b.
%= (Residuo y asigna): a %= b obtiene el sobrante de una división y lo asigna.
**= (Potencia y asigna): a **= b eleva la variable a una potencia y guarda el resultado.
2. Operadores de Comparación
Se usan para comparar dos valores y siempre devuelven un resultado de Verdadero o Falso (Booleano).
== (Igual a): Compara si dos valores son iguales.
< (Menor que): Revisa si el valor de la izquierda es más pequeño.
> (Mayor que): Revisa si el valor de la izquierda es más grande.
<= (Menor o igual a): Revisa si es menor o si son exactamente iguales.
>= (Mayor o igual a): Revisa si es mayor o si son exactamente iguales.
3. Operadores Lógicos
Sirven para combinar varias comparaciones al mismo tiempo.
&& (AND / Y): Devuelve verdadero solo si todas las condiciones se cumplen.
|| (OR / O): Devuelve verdadero si al menos una de las condiciones se cumple.
! (NOT / NO): Invierte el resultado (si algo es verdadero, lo vuelve falso).
4. Operador de Cadena de Texto
Concatenación: Es la acción de "pegar" dos o más textos. Normalmente se usa el símbolo +.
Ejemplo: "Hola" + " Mundo" da como resultado "Hola Mundo".
5. Operador Ternario
Es una forma corta de escribir un if-else (una decisión) en una sola línea. Funciona como una pregunta rápida:
Estructura: condición ? resultado_si_es_cierto : resultado_si_es_falso
Cómo funciona:
Se evalúa la condición.
Si es cierta, se ejecuta lo que está después del signo ?.
Si es falsa, se ejecuta lo que está después de los :.
Ejemplo: edad >= 18 ? "Mayor de edad" : "Menor de edad"
